Binomial Formula

\[P(x) = \frac{n!}{!x * (n-x)!} *p^{x} * q^{n-x}\]

The Binomial Formula is used to gauge the probability of an event to occur given a number of trials
\(x\) = the number of successful trials
\(n\) = the number of trials
\(p\) = the probability of an event occuring in a trial
\(q\) = the probability of the secondary event occuring in a trial. (Inverse of \(p\))
\(p(x)\) = the probability of \(x\) occuring exactly \(x\) times
